*Auto-generate professional contracts with risk detection and compliance checking.*

An AI-powered legal document assistant that drafts, reviews, and customizes business contracts. From freelance service agreements and NDAs to licensing deals and partnership contracts, get professionally structured legal documents tailored to your specific needs.

## Features

| Feature | Description |
|---------|-------------|
| Contract Drafting | Generates professional contracts from templates covering 20+ common business scenarios |
| Risk Detection | Scans contracts for unfavorable clauses, missing protections, and potential liabilities |
| Compliance Checker | Ensures contracts align with relevant regulations and industry standards |
| Clause Library | Provides a comprehensive library of pre-vetted clauses for customization |
| Multi-Party Support | Handles complex agreements with multiple parties, roles, and obligations |
| Version Comparison | Tracks changes between contract versions and highlights key differences |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-contract-generator-skill
```

## Usage

```
generate [contract type]   # Draft a new contract from template
review [contract]          # Analyze a contract for risks and issues
customize [clause]         # Modify specific clauses to your needs
compare [v1 vs v2]         # Compare two contract versions
checklist [contract type]  # Generate compliance checklist
```
